# Peckocko #

## Local Installation ##

1. Clone the repository
2. Run `nvm use 12.14.0`
3. Run `npm install`
5. Run `npm run start:api` -> localhost:3000
4. Run `npm run start:app` ->

## Usage ##

You can now access the application at `http://localhost:4200/`
